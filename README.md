# 🩺 MediQueue (Frontend) — Tutor Booking Platform

MediQueue is a dedicated tutor booking web application engineered to bridge the gap between students and medical/healthcare tutors. The platform streamlines the learning process by eliminating manual scheduling, preventing time slot conflicts, and organizing upcoming study sessions efficiently.



---

## 🚀 Key Features

* **Dynamic Tutor Browsing & Filtering:** Students can effortlessly explore available educators and filter profiles instantly based on subject expertise, pricing, and specific time availability.
* **Conflict-Free Interactive Booking:** An automated scheduling system locks down available hours, preventing overlapping appointments and double-bookings.
* **Digital Session Token Generation:** Upon successful booking, the platform automatically generates unique digital session tokens for students to securely access their upcoming classes.
* **Comprehensive User Dashboard:** A centralized, intuitive panel for students and tutors to manage scheduled classes, monitor upcoming sessions, and review historical bookings.
* **Secure Authentication & Session Management:** Integrated secure user registration and login pathways, ensuring student data privacy and personalized schedules.

---

## 🛠️ Tech Stack

* **Framework:** Next.js 15 (App Router & Turbopack)
* **Styling:** Tailwind CSS
* **State Management / Data Fetching:** React Hooks & Context API / Axios
* **Icons:** React Icons / Lucide React
